# Example NextJS App
a test project app for https://www.id-sign.com/
## Stack
- NextJs
- GraphQL
- React-Bootstrap

## Getting Started

For dev server run:

```bash
npm run dev
```

For production server:
```bash
npm run build && npm run start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
